Planning: What type of structure did you choose to implement?

I created a html file for each page, in addition to that I have also created an internal css file to control the design of all my html files and I also added an js file to write bits of code to apply to the website.
I made sure to use a large enough font sizes in all my pages so people without a perfect view can easily see eveything.
I also tagged images with a name in case the images don't load up.
Another thing I did is I sticked to the same colour for most of the website so it wouldn't be a challenge to see clearly for daltonic people and not to confuse them.
First thing I beared in mind in terms of usuability was the navegation bar I included this bar in all the pages and in the same place so people can always go to the pages they want instead of having to look for a link to go the wanted page.
I also implemented an "read more" button so people don't get too much information just when they open the page.
I also linked the heading of the website "Daily left" to homepage, so it's easier for someone who is not used to use the internet to control the website.
